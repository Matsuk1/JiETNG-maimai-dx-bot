# 隐私政策

最后更新：2025年1月

## 概述

JiETNG 重视您的隐私。本隐私政策解释了我们如何收集、使用和保护您的个人信息。

## 我们收集的信息

### 必需信息

为了提供服务，JiETNG 收集以下信息：

- **SEGA ID 凭据**：您的 SEGA ID 用户名和密码
- **LINE 用户 ID**：用于识别您的 LINE 账户
- **maimai 成绩数据**：从官方 maimai NET 获取的游戏数据

### 自动收集的信息

- **命令使用记录**：您使用的命令和使用时间
- **错误日志**：用于调试和改进服务的错误信息

## 我们如何使用您的信息

我们使用收集的信息来：

1. **提供服务**：
   - 从 maimai NET 获取您的成绩数据
   - 生成成绩图表和统计信息
   - 响应您的命令

2. **改进服务**：
   - 分析使用模式以改进功能
   - 调试和修复错误
   - 优化性能

3. **安全目的**：
   - 防止滥用和欺诈
   - 保护服务的完整性

## 数据存储和安全

### 加密

- **密码加密**：您的 SEGA 密码使用行业标准加密方法存储
- **传输安全**：所有数据传输都使用 HTTPS 加密

### 数据保留

- **活跃账户**：只要您的账户处于活跃状态，我们就会保留您的数据
- **删除账户**：使用 `unbind` 命令时，所有存储的数据将被永久删除

### 访问控制

- 只有 JiETNG 系统可以访问您的数据
- 没有人工查看或访问您的个人信息
- 我们不会与第三方分享您的数据

## 第三方服务

JiETNG 与以下第三方服务交互：

- **SEGA maimai NET**：获取您的游戏数据
- **LINE Platform**：消息传递功能

我们不控制这些第三方的隐私政策，请查看他们的隐私政策以了解详情。

## 您的权利

您对您的数据拥有以下权利：

### 访问权

- 使用 `get me` 命令查看存储的账户信息

### 删除权

- 使用 `unbind` 命令删除所有存储的数据
- 删除是立即且永久的

### 更新权

- 使用 `bind` 命令更新您的凭据
- 使用 `maimai update` 命令刷新您的成绩数据

## Cookies 和跟踪

JiETNG 不使用 cookies 或其他跟踪技术。

## 未成年人隐私

JiETNG 不会故意收集 13 岁以下儿童的信息。如果您认为我们可能收集了未成年人的信息，请联系我们。

## 数据泄露通知

如果发生影响您数据的安全泄露，我们将：

1. 立即调查事件
2. 采取措施减轻损害
3. 尽快通知受影响的用户

## 政策变更

我们可能会不时更新本隐私政策。重大变更将通过以下方式通知：

- 在机器人中发布公告
- 更新本页面的"最后更新"日期

## 数据可移植性

您可以要求导出您的数据。请联系支持团队以请求数据导出。

## 法律依据

我们根据以下法律依据处理您的数据：

- **合同必要性**：提供您请求的服务
- **合法利益**：改进和保护我们的服务
- **同意**：您使用我们的服务即表示同意本政策

## 国际数据传输

您的数据可能会在您所在国家/地区以外的服务器上处理。我们采取适当措施确保您的数据受到保护。
