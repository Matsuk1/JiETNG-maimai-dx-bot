# 成绩系统

Best 50（b50）和 Best 100（b100）图表是 JiETNG 的旗舰功能，以精美的可视化设计展示您的最高评分成绩。

## 什么是 Best 50？

"Best 50"系统是 Maimai DX 的官方排名方法，由以下组成：

- **Best 35**：您在**旧曲（以前版本的歌曲）**中的前 35 个最高分
- **Best 15**：您在**新曲（当前版本的歌曲）**中的前 15 个最高分

您的 **Maimai DX Rating** 是这 50 个分数的总和。

## 命令

### 基础 Best 图表

```
b50          # 生成 Best 50 图表
b100         # 生成 Best 100 图表
best50       # b50 的替代命令
best100      # b100 的替代命令
```

### 变体

```
best35       # 仅显示前 35 首旧曲
best15       # 仅显示前 15 首新曲
ab50         # All Best 50（忽略歌曲版本）
ab35         # All Best 35
apb50        # All Perfect Best 50（仅 AP/AP+ 成绩）
idlb50       # Ideal Best 50（模拟 SSS+ 成绩）
```

## 图表功能

### 显示内容

每个成绩卡片显示：

- 🎵 **歌曲名称**和难度
- 🎮 **歌曲版本**：旧曲或新曲
- ⭐ **内部等级**：实际难度（例如 14.7）
- 💯 **达成率**：您的分数百分比
- 🏆 **评分等级**：SSS+、SSS、SS+ 等
- 🌟 **Full Combo 类型**：AP+、AP、FC+、FC
- 🔥 **Full Sync**：FDX+、FDX、FS+、FS
- 📊 **Rating**：单曲评分（0-22.4）
- 📈 **DX Score**：您的 DX 分数 / 最大可能分数

### 用户信息头部

图表包含您的个人资料：

- 👤 玩家名称和头像
- 🏅 段位/等级
- ⭐ 总 rating
- 🎖️ 称号牌
- 📅 最后更新时间

## 高级用法

### 过滤成绩

您可以应用过滤器来自定义 b50 输出：

#### 按等级过滤

```
b50 -lv 15              # 仅等级 15 的歌曲
b50 -lv 14 15           # 等级 14-15
b50 -lv 13+             # 等级 13.7+ 及以上
```

#### 按 Rating 过滤

```
b50 -ra 200             # 仅 Rating 200+
b50 -ra 180 200         # Rating 180-200
```

#### 按达成率过滤

```
b50 -scr 100.5          # 达成率 100.5%+
b50 -scr 99 100         # 达成率 99%-100%
```

#### 按 DX Score 过滤

```
b50 -dx 95              # DX score 95%+
b50 -dx 90 95           # DX score 90-95%
```

### 组合过滤器

您可以组合多个过滤器：

```
b50 -lv 15 -scr 100.5            # 等级 15 且达成率 100.5%+ 的歌曲
b50 -ra 200 -dx 95               # Rating 200+ 且 DX score 95%+
b50 -lv 14 15 -scr 99.5 -dx 90   # 复杂过滤
```

## 图表类型说明

### Best 50 (b50)

遵循官方规则的标准排名图表：
- 标准歌曲前 35 名
- DX 歌曲前 15 名
- 您的实际 Maimai DX rating

**使用场景**：查看您的官方 rating 和进度

### Best 100 (b100)

显示更多成绩的扩展版本：
- 标准歌曲前 70 名
- DX 歌曲前 30 名
- 查看超出 rating 范围的歌曲

**使用场景**：找到刚好低于您的 b50 门槛的歌曲

### All Best 50 (ab50)

忽略歌曲类型区分：
- 不论类型的前 50 个最高分
- 可能全部是标准歌曲或全部是 DX 歌曲
- 展示您的绝对最佳表现

**使用场景**：查看不分类别的最高成就

### All Perfect Best 50 (apb50)

仅显示完美演奏：
- 仅 AP（All Perfect）和 AP+ 成绩
- 按 rating 排名
- 追踪您的完美 combo

**使用场景**：监控您的完美演奏进度

### Ideal Best 50 (idlb50)

理论最大 rating：
- 模拟所有歌曲的 SSS+（100.5%+）
- 显示潜在的 rating 增长
- 激励改进

**使用场景**：为 rating 提升设定目标

## 理解输出

### Rating 计算

每首歌曲的 rating 计算如下：

```
基础 Rating = 内部等级 × 10
达成率修正值 = 基于您的达成率 %

最终 Rating = 基础 Rating + 达成率修正值
```

达成率修正值：
- **100.5000%+**：+2.0 rating (SSS+)
- **100.0000%**：+1.5 rating (SSS)
- **99.5000%+**：+1.0 rating (SS+)
- **99.0000%+**：+0.5 rating (SS)
- **98.0000%+**：+0.0 rating (S+)
- **97.0000%+**：-1.0 rating (S)
- 低于 97%：更低的 rating

### 颜色编码

图表使用颜色表示表现：
- 🟣 **紫色**：SSS+ (100.5%+)
- 🔴 **红色**：SSS/SS+ (99.5-100.49%)
- 🟠 **橙色**：SS/S+ (98-99.49%)
- 🟡 **黄色**：S (97-97.99%)
- ⚪ **灰色**：低于 S

## 技巧与窍门

### 最大化您的 Rating

1. **专注于高内部值歌曲**：等级 14.9、15.0 谱面
2. **改进 SSS → SSS+**：每首歌曲 +0.5 rating
3. **提升 SS+ → SSS**：+0.5 rating 增益
4. **替换低 rating 的 b50 歌曲**：查看 b100 寻找机会

### 寻找改进目标

使用过滤器找到具体目标：

```
b100 -scr 99.5 100.49    # 找到 SSS 歌曲提升至 SSS+
b100 -lv 15              # 查看所有等级 15 的演奏
```

比较 b50 与 ab50 以查看类型不平衡：

```
b50     # 官方 rating
ab50    # 整体最佳成绩
```

如果您的 ab50 明显更高，请专注于代表性不足的歌曲类型。

## 常见问题

### 为什么我的 rating 与游戏内不同？

- JiETNG 仅在您运行 `maimai update` 时更新
- 官方 rating 在每次游玩后更新
- 定期同步您的成绩以保持最新

### 有些成绩缺失？

确保您最近玩过这些歌曲。较旧的成绩可能不会出现，如果：
- 它们已被更好的成绩替换
- 歌曲已从游戏中移除
- 数据同步问题（再次尝试 `maimai update`）

### 我可以查看历史 b50 吗？

目前，JiETNG 显示您当前的 b50。历史跟踪计划在未来更新中推出。
