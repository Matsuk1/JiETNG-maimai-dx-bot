# 基础命令

本页面涵盖您在使用 JiETNG 时会经常用到的所有基本命令。

## 账户管理

### 绑定 SEGA 账户

链接您的 SEGA ID 以开始使用 JiETNG：

```
bind
```

这将提供一个用于安全绑定的网页链接。

### 解绑账户

移除您的 SEGA ID 并删除所有存储的数据：

```
unbind
```

:::danger 警告
此操作不可逆。您的所有数据将被永久删除。
:::

### 更新成绩

从 SEGA 获取您的最新成绩：

```
maimai update
update
マイマイアップデート
レコードアップデート
```

**速率限制**：每 5 分钟一次

## 查看成绩

### Best 图表

生成您的 Best 50 或 Best 100：

```
b50                # Best 50（35 首标准 + 15 首 DX）
b100               # Best 100（70 首标准 + 30 首 DX）
best50             # 与 b50 相同
best100            # 与 b100 相同
```

### Best 变体

```
best35             # 仅前 35 首标准歌曲
best15             # 仅前 15 首 DX 歌曲
ab50               # All Best 50（忽略类型）
ab35               # All Best 35
apb50              # All Perfect Best 50（仅 AP/AP+）
idlb50             # Ideal Best 50（模拟 SSS+）
```

### 最近游玩

查看您最近的 50 次游玩：

```
recent
rct50
```

## 歌曲搜索

### 按名称搜索

通过标题查找歌曲：

```
search <歌曲名称>
```

示例：
```
search 檄
search geki
search shama
```

**支持：**
- 完整歌曲名称（日文、英文、中文）
- 部分名称
- 罗马音
- 常用缩写

### 按歌曲 ID 搜索

使用官方歌曲 ID：

```
id:<song_id>
```

示例：
```
id:11051
```

## 歌曲信息

### 查看歌曲详情

获取歌曲的全面信息：

```
<歌曲缩写>
```

示例：
```
geki              # 显示 檄・GEKI・GEKI 信息
ifuudoudou        # 显示 イフウドウドウ 信息
```

### 查看您的成绩

如果您玩过该歌曲，将显示您的成绩。如果没有，则显示歌曲信息。

## 基于等级的查询

### 查看某个等级的所有成绩

```
lv<等级>
lv<等级> <页码>
```

示例：
```
lv15              # 您的所有等级 15 成绩（第 1 页）
lv14+ 2           # 等级 14.7+ 成绩（第 2 页）
lv13              # 等级 13.0-13.6 成绩
```

**等级格式**：
- `15` = 15.0
- `15+` = 15.7+
- `14` = 14.0-14.6
- `14+` = 14.7-14.9

### 分页

每页最多显示 50 个成绩。使用页码查看更多：

```
lv15 1            # 第 1 页
lv15 2            # 第 2 页
lv15 3            # 第 3 页
```

## 牌子进度

追踪您的牌子完成进度：

```
<版本><牌子类型>
```

示例：
```
真極              # BUDDiES 極 牌子
真将              # BUDDiES 将 牌子
真神              # BUDDiES 神 牌子
真舞舞            # BUDDiES 舞舞 牌子（Sync 牌子）
```

**版本缩写**：
- `真` = BUDDiES
- `祭` = FESTiVAL+
- `宴` = FESTiVAL
- `舞` = UNiVERSE+
- `双` = UNiVERSE

**牌子类型**：
- `極` = 所有 Master 谱面 FC+ 或更好
- `将` = 所有 Master 谱面 SSS 或更好
- `神` = 所有 Master 谱面 AP 或更好
- `舞舞` = 所有 Master 谱面 FDX 或更好

## 随机歌曲

获取随机歌曲推荐：

```
random
ランダム曲
```

**使用过滤器**：
```
random <等级>            # 特定等级的随机歌曲
random lv14+              # 随机等级 14.7+ 歌曲
```

## 计算器

### 达成率计算器

计算达到目标达成率所需的百分比：

```
calc <tap> <hold> <slide> <touch> <break>
```

示例（100 个 tap、50 个 hold、30 个 slide、20 个 touch、10 个 break 的歌曲）：
```
calc 100 50 30 20 10
```

显示每种音符类型的达成率值。

## 用户资料

### 查看 Maipass

生成带有二维码的玩家卡片：

```
maipass
pass
```

这将创建一个可分享的卡片，包含：
- 您的用户名和 rating
- 用于添加好友的二维码
- 个人资料图标和牌子

### 获取用户信息

```
me
profile
```

显示您当前的：
- Rating
- 段位/等级
- 最后更新时间
- 绑定状态

## 帮助与信息

### 查看帮助

```
help
ヘルプ
```

显示快速命令参考。

### 查看 Bot 信息

```
about
info
```

显示 bot 版本、运行时间和统计信息。

## 语言

### 更改语言

```
lang ja           # 日本語
lang en           # English
lang zh           # 中文
```

所有未来的消息将使用您选择的语言。

## 提示

### 命令快捷方式

许多命令有多个别名：

```
b50 = best50
b100 = best100
lv = level
```

### 不区分大小写

命令不区分大小写：

```
B50 = b50 = Best50
RANDOM = random
```

### 空格

大多数命令可以优雅地处理额外的空格：

```
search  geki     # 正常工作
b50              # 不需要空格
lv 15            # 空格可选
```
