# 基本コマンド

このページでは、JiETNG を使用する際に頻繁に使用する基本的なコマンドをすべて紹介します。

## アカウント管理

### SEGA アカウントを連携

SEGA ID をリンクして JiETNG の使用を開始：

```
bind
```

安全な連携用のウェブリンクが提供されます。

### アカウント連携を解除

SEGA ID を削除し、保存されたすべてのデータを削除：

```
unbind
```

:::danger 警告
この操作は取り消せません。すべてのデータが永久に削除されます。
:::

### スコアを更新

SEGA から最新のスコアを取得：

```
maimai update
update
マイマイアップデート
レコードアップデート
```

## 計算機

### 達成率計算機

目標達成率に到達するために必要なパーセンテージを計算：

```
calc <tap> <hold> <slide> [<touch>] <break>
```

例（100 tap、50 hold、30 slide、20 touch、10 break の楽曲）：
```
calc 100 50 30 20 10
```

各ノートタイプの達成率の値を表示します。

## ユーザープロファイル

### Maipass を表示

QR コード付きのプレイヤーカードを生成：

```
maipass
pass
```

これにより共有可能なカードが作成されます：
- ユーザー名とレーティング
- フレンド追加用の QR コード
- プロフィールアイコンとプレート

### ユーザー情報を取得

```
getme
```

## ヘルプと情報

### ヘルプを表示

```
help
ヘルプ
```

クイックコマンドリファレンスを表示します。

## ヒント

### コマンドのショートカット

多くのコマンドには複数のエイリアスがあります：

```
b50 = best50
b100 = best100
```

### 大文字小文字を区別しない

コマンドは大文字小文字を区別しません：

```
B50 = b50 = Best50
RANDOM = random
```

### スペース

ほとんどのコマンドは余分なスペースを適切に処理します：

```
search  geki     # 正常に動作
b50              # スペース不要
```
