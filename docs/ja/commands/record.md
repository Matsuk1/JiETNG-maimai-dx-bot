# スコアシステム

Best 50（b50）と Best 100（b100）チャートは JiETNG の基本機能で、美しいビジュアルデザインで最高評価のスコアを表示します。

## Best 50 とは？

「Best 50」システムは Maimai でらっくす の公式ランキング方法で、以下で構成されます：

- **Best 35**：**旧曲**（以前のバージョンの楽曲）でのトップ 35 のハイスコア
- **Best 15**：**新曲**（現在のバージョンの楽曲）でのトップ 15 のハイスコア

あなたの **Maimai でらっくす Rating** は、これら 50 のスコアの合計です。

## コマンド

### 基本 Best チャート

```
b50          # Best 50 チャートを生成
b100         # Best 100 チャートを生成
best50       # b50 の代替コマンド
best100      # b100 の代替コマンド
```

### バリエーション

```
best35       # トップ 35 旧曲のみを表示
best15       # トップ 15 新曲のみを表示
ab35         # All Best 35 （楽曲バージョンを無視）
ab50         # All Best 50 （楽曲バージョンを無視）
ab100        # All Best 100（楽曲バージョンを無視）
ab200        # All Best 200（楽曲バージョンを無視）
apb50        # All Perfect Best 50（AP/AP+ スコアのみ）
idlb50       # Ideal Best 50（最高スコアをシミュレート）
```

## チャート機能

### 表示内容

各スコアカードには以下が表示されます：

- 🎵 **楽曲名**と難易度
- 🎮 **楽曲バージョン**：旧曲または新曲
- ⭐ **内部定数**：例：14.7
- 💯 **達成率**：スコアのパーセンテージ
- 🏆 **評価ランク**：SSS+、SSS、SS+ など
- 🌟 **Full Combo タイプ**：AP+、AP、FC+、FC
- 🔥 **Full Sync**：Fでらっくす+、Fでらっくす、FS+、FS
- 📊 **Rating**：単一楽曲の評価
- 📈 **でらっくす Rating**：あなたの でらっくす Rating

### ユーザー情報ヘッダー

チャートにはプロフィールが含まれます：

- 👤 プレイヤー名とアバター
- 🏅 段位/ランク
- ⭐ 総レーティング
- 🎖️ 称号

## 高度な使用方法

### スコアのフィルタリング

b50 出力をカスタマイズするためにフィルターを適用できます：

#### レベルでフィルター

```
b50 -lv 15              # レベル 15 の楽曲のみ
b50 -lv 14 15           # レベル 14-15
b50 -lv 13+             # レベル 13.7+ 以上
```

#### Rating でフィルター

```
b50 -ra 200             # Rating 200+ のみ
b50 -ra 180 200         # Rating 180-200
```

#### 達成率でフィルター

```
b50 -scr 100.5          # 達成率 100.5%+
b50 -scr 99 100         # 達成率 99%-100%
```

#### でらっくす Score でフィルター

```
b50 -dx 95              # でらっくす score 95%+
b50 -dx 90 95           # でらっくす score 90-95%
```

#### バージョンでフィルター

```
b50 -ver buddies                   # Buddies バージョンのみ
b50 -ver splash splash+            # Splash と Splash PLUS バージョン
b50 -ver festival+ buddies         # FESTiVAL PLUS と Buddies バージョン
```

::: tip バージョン名の注意事項
- バージョン名は大文字小文字を区別しません
- PLUS バージョンを示すには `+` を使用します（例：`splash+`）
- 複数のバージョンをスペースで区切って指定できます
:::

### フィルターの組み合わせ

複数のフィルターを組み合わせることができます：

```
b50 -lv 15 -scr 100.5                    # レベル 15 かつ達成率 100.5%+ の楽曲
b50 -ra 200 -dx 95                       # Rating 200+ かつ でらっくす score 95%+
b50 -ver buddies -lv 14                  # Buddies バージョンかつレベル ≥14 の楽曲
b50 -ver splash splash+ -scr 100         # Splash/Splash+ かつ達成率 ≥100% の楽曲
b50 -lv 14 15 -scr 99.5 -dx 90           # 複雑なフィルター
```

## チャートタイプの説明

### Best 50 (b50)

公式ルールに従った標準ランキングチャート：
- 旧バージョン楽曲のトップ 35
- 新バージョン楽曲のトップ 15
- あなたの実際の でらっくす rating

**使用シーン**：公式 rating と進捗を確認

### Best 100 (b100)

より多くのスコアを表示する拡張版：
- 旧バージョン楽曲のトップ 70
- 新バージョン楽曲のトップ 30

**使用シーン**：b50 のしきい値をわずかに下回る楽曲を見つける

### All Best 50 (ab50)

楽曲バージョンの区別を無視：
- バージョンに関係なくトップ 50 のハイスコア

**使用シーン**：バージョンを問わない最高成績を確認

### All Perfect Best 50 (apb50)

AP 楽曲のみを表示：
- AP（All Perfect）と AP+ スコアのみ
- rating 順にランク付け

**使用シーン**：AP 進捗を監視

### Ideal Best 50 (idlb50)

理論上の最大 rating：
- すべての楽曲で上位階層のスコアをシミュレート
- 潜在的な rating 成長を表示

**使用シーン**：rating 向上の目標を設定

## よくある質問

### なぜ私の rating がゲーム内と異なるのですか？

- JiETNG は `maimai update` を実行したときのみ更新されます
- 一部の楽曲定数が標準ではありません

### 一部のスコアが欠けていますか？

最近これらの楽曲をプレイしたか確認してください。古いスコアは以下の場合表示されない可能性があります：
- より良いスコアに置き換えられた
- 楽曲がゲームから削除された
- データ同期の問題（再度 `maimai update` を試してください）


## 定数クエリ

指定した難易度レベルのすべての楽曲を、内部定数（13.0、13.1、13.2 など）でグループ化して表示します。

### コマンド形式

```
13の定数リスト    # 日本語コマンド（定数リスト）
13のレベルリスト  # 日本語コマンド（レベルリスト）
13 level-list   # 英語コマンド
```

### 表示内容

- 左側に内部定数（13.0、13.1、13.2 など）を表示
- 右側に各定数のすべての楽曲カバーを表示
- 上部に総楽曲数の統計を表示
- 定数は高いものから低いものへとソート

### サーバー選択

レベルクエリは現在選択されているサーバー（JP または INTL）を自動的に使用します：
- JP サーバーの楽曲を表示するには `maimai jp` を使用
- INTL サーバーの楽曲を表示するには `maimai intl` を使用

### 使用シーン

- 📋 **定数分布を確認**：難易度の定数範囲を理解する
- 🎯 **目標楽曲を見つける**：特定の定数範囲で楽曲を見つける
- 📊 **難易度を比較**：異なるレベルの楽曲数を比較する
