# スコアシステム

Best 50（b50）と Best 100（b100）チャートは JiETNG の基本機能で、美しいビジュアルデザインで最高評価のスコアを表示します。

## Best 50 とは？

「Best 50」システムは Maimai DX の公式ランキング方法で、以下で構成されます：

- **Best 35**：**旧曲（以前のバージョンの楽曲）**でのトップ 35 のハイスコア
- **Best 15**：**新曲（現在のバージョンの楽曲）**でのトップ 15 のハイスコア

あなたの **Maimai DX Rating** は、これら 50 のスコアの合計です。

## コマンド

### 基本 Best チャート

```
b50          # Best 50 チャートを生成
b100         # Best 100 チャートを生成
best50       # b50 の代替コマンド
best100      # b100 の代替コマンド
```

### バリエーション

```
best35       # トップ 35 旧曲のみを表示
best15       # トップ 15 新曲のみを表示
ab50         # All Best 50（楽曲バージョンを無視）
ab35         # All Best 35（楽曲バージョンを無視）
apb50        # All Perfect Best 50（AP/AP+ スコアのみ）
idlb50       # Ideal Best 50（最高スコアをシミュレート）
```

## チャート機能

### 表示内容

各スコアカードには以下が表示されます：

- 🎵 **楽曲名**と難易度
- 🎮 **楽曲バージョン**：旧曲または新曲
- ⭐ **内部定数**：例：14.7
- 💯 **達成率**：スコアのパーセンテージ
- 🏆 **評価ランク**：SSS+、SSS、SS+ など
- 🌟 **Full Combo タイプ**：AP+、AP、FC+、FC
- 🔥 **Full Sync**：FDX+、FDX、FS+、FS
- 📊 **Rating**：単一楽曲の評価
- 📈 **DX Rating**：あなたの DX Rating

### ユーザー情報ヘッダー

チャートにはプロフィールが含まれます：

- 👤 プレイヤー名とアバター
- 🏅 段位/ランク
- ⭐ 総レーティング
- 🎖️ 称号

## 高度な使用方法

### スコアのフィルタリング

b50 出力をカスタマイズするためにフィルターを適用できます：

#### レベルでフィルター

```
b50 -lv 15              # レベル 15 の楽曲のみ
b50 -lv 14 15           # レベル 14-15
b50 -lv 13+             # レベル 13.7+ 以上
```

#### Rating でフィルター

```
b50 -ra 200             # Rating 200+ のみ
b50 -ra 180 200         # Rating 180-200
```

#### 達成率でフィルター

```
b50 -scr 100.5          # 達成率 100.5%+
b50 -scr 99 100         # 達成率 99%-100%
```

#### DX Score でフィルター

```
b50 -dx 95              # DX score 95%+
b50 -dx 90 95           # DX score 90-95%
```

### フィルターの組み合わせ

複数のフィルターを組み合わせることができます：

```
b50 -lv 15 -scr 100.5            # レベル 15 かつ達成率 100.5%+ の楽曲
b50 -ra 200 -dx 95               # Rating 200+ かつ DX score 95%+
b50 -lv 14 15 -scr 99.5 -dx 90   # 複雑なフィルター
```

## チャートタイプの説明

### Best 50 (b50)

公式ルールに従った標準ランキングチャート：
- 旧バージョン楽曲のトップ 35
- 新バージョン楽曲のトップ 15
- あなたの実際の DX rating

**使用シーン**：公式 rating と進捗を確認

### Best 100 (b100)

より多くのスコアを表示する拡張版：
- 旧バージョン楽曲のトップ 70
- 新バージョン楽曲のトップ 30

**使用シーン**：b50 のしきい値をわずかに下回る楽曲を見つける

### All Best 50 (ab50)

楽曲バージョンの区別を無視：
- バージョンに関係なくトップ 50 のハイスコア

**使用シーン**：バージョンを問わない最高成績を確認

### All Perfect Best 50 (apb50)

AP 楽曲のみを表示：
- AP（All Perfect）と AP+ スコアのみ
- rating 順にランク付け

**使用シーン**：AP 進捗を監視

### Ideal Best 50 (idlb50)

理論上の最大 rating：
- すべての楽曲で上位階層のスコアをシミュレート
- 潜在的な rating 成長を表示

**使用シーン**：rating 向上の目標を設定

## よくある質問

### なぜ私の rating がゲーム内と異なるのですか？

- JiETNG は `maimai update` を実行したときのみ更新されます
- 一部の楽曲定数が標準ではありません

### 一部のスコアが欠けていますか？

最近これらの楽曲をプレイしたか確認してください。古いスコアは以下の場合表示されない可能性があります：
- より良いスコアに置き換えられた
- 楽曲がゲームから削除された
- データ同期の問題（再度 `maimai update` を試してください）
