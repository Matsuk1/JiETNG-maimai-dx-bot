# クイックスタート

3つの簡単なステップで JiETNG を使って maimai DX のスコア追跡を始めましょう。

## 前提条件

始める前に、以下を用意してください：

- ✅ LINE アカウント
- ✅ SEGA ID アカウント（maimai NET 用）
- ✅ maimai NET へのアクセス権限
- ✅ スマートフォンまたはコンピューター

## ステップ 1: ボットを追加

1. LINE で **@299bylay** を検索
2. または QR コードをスキャン：[LINE Bot QR Code]
3. 「友だち追加」をクリック
4. 会話を開始

## ステップ 2: SEGA ID を連携

これが最も重要なステップです - SEGA アカウントをボットに接続します。

### 連携を開始

ボットに送信：

```
bind
```

### 連携手順を完了

1. ボットが **連携 URL** ボタンを送信します
2. **ボタンをクリック**して連携ページを開く
3. **認証情報を入力**：
   - SEGA ID（ユーザー名）
   - パスワード
   - バージョンを選択（JP または International）
   - 言語を選択
4. **フォームを送信**
5. 確認メッセージを待つ

:::warning ⚠️ 重要なセキュリティ注意事項
- **チャットでパスワードを入力しないでください**
- ボットが提供する公式リンクのみを使用
- トークンは 2 分後に期限切れになります
- パスワードは暗号化されて保存されます
:::

### 連携を確認

連携が成功したか確認：

```
get me
```

SEGA ID、バージョン、言語情報が表示されるはずです。

[連携にヘルプが必要ですか？詳細な連携ガイドをご覧ください →](/ja/guide/binding)

## ステップ 3: スコアを更新

maimai NET からスコアを同期しましょう！

### 初回同期

```
maimai update
```

### 処理を待つ

- ⏱️ 更新時間：20-30 秒
- 📊 すべての楽曲とスコアを取得

:::tip 🎯 いつ更新すべきか
- ゲームセンターでプレイした後
- 最新のスコアを確認したい時
:::

## ステップ 4: Best 50 を生成

ここからが楽しい部分です - 最高スコアを確認しましょう！

### 基本コマンド

```
b50
```

### 表示される内容

美しいチャートが表示されます：
- 📊 トップ 35 の旧曲（以前のバージョンの楽曲）
- 🎵 トップ 15 の新曲（現在のバージョンの楽曲）
- 📈 総レーティング
- 🎯 各楽曲の詳細情報

### その他のバリエーション

```
b100      # Best 100（トップ 70 旧曲 + トップ 30 新曲）
b35       # トップ 35 旧曲のみ
b15       # トップ 15 新曲のみ
```

[Best 50 の詳細 →](/ja/commands/record)

## 次にすべきこと

### 機能を探索

セットアップが完了したので、これらのコマンドを試してみましょう：

**楽曲を検索**：
```
[曲名] info
```

**ランダム楽曲**：
```
random
random 14
```

**フレンドリスト**：
```
friend list
```

**バージョン達成度を表示**：
```
FESTiVAL achievement
```

## よく使うコマンド一覧

| コマンド | 用途 |
|----------|---------|
| `maimai update` | maimai NET からスコアを同期 |
| `b50` | Best 50 チャートを生成 |
| `[曲名] song-info` | 楽曲情報を検索 |
| `[曲名] record` | その楽曲のスコアを表示 |
| `14 record-list` | すべてのレベル 14 のスコアを表示 |
| `friend list` | フレンドを表示 |
| `get me` | アカウント情報を表示 |
| `unbind` | SEGA ID の連携を解除 |

## トラブルシューティング

### 「SEGA ID が連携されていません」

**解決方法**：ステップ 2 を完了 - `bind` コマンドを使用

### 「更新に失敗しました」

**考えられる原因**：
- maimai NET がメンテナンス中
- ネットワーク接続の問題
- SEGA 認証情報が正しくない

**解決方法**：
- 数分待ってから再試行
- maimai NET に直接アクセスできるか確認
- `unbind` してから再度 `bind`

### 「コマンドが機能しない」

**チェックリスト**：
- ✅ 連携済みですか？（`get me`）
- ✅ スコアは更新済みですか？（`maimai update`）
- ✅ スペルは正しいですか？
- ✅ 正しいコマンドを使っていますか？

[完全なトラブルシューティングガイドを見る →](/ja/more/faq)

## ヒントとコツ

### 💡 プロのヒント

1. **定期的に更新**：プレイ後すぐに `maimai update`
2. **フィルターを使用**：`b50 -lv 14 -scr 97 98` で改善機会を見つける
3. **フレンドを追加**：他のプレイヤーとスコアを比較
4. **プレートを追跡**：`[バージョン] achievement` で進捗を追跡
5. **目標を設定**：`idealb50` で潜在的なレーティングを確認
