# よくある質問

JiETNG の使用に関するよくある質問と回答です。

## はじめに

### JiETNG とは何ですか？

JiETNG は 『maimai でらっくす』 プレイヤー向けの LINE ボットで、以下の機能を提供します：
- スコア追跡と分析
- Best 50 チャート生成
- 楽曲検索と発見
- プレート進捗追跡
- フレンドランキング比較
- その他多数！

### JiETNG は有料ですか？

いいえ！JiETNG は完全に**無料**でご利用いただけます。ただし、[寄付](/ja/more/support)を通じて開発を支援することができます。


### SEGA ID は必要ですか？

はい、ほとんどの機能を使用するには以下が必要です：
- SEGA IDアカウント
- maimai NET（オンラインスコア追跡）へのアクセス
- アカウントをボットに連携

セットアップ手順については、[アカウント連携](/ja/guide/binding)を参照してください。

## アカウント連携

### SEGA ID を連携するにはどうすればよいですか？

1. ボットに `bind` を送信
2. 連携 URL ボタンをクリック
3. Web フォームで SEGA ID とパスワードを入力
4. サーバーバージョンを選択
5. 連携を確認

**重要**: チャットで認証情報を入力しないでください。連携は Web 経由でのみ行われます。

詳細は[アカウント連携ガイド](/ja/guide/binding)を参照してください。

### チャットで SEGA 認証情報を入力できますか？

**いいえ。** セキュリティ上の理由から、JiETNG は Web ベースの連携のみを使用します。チャットでパスワードを入力しないでください。

### 連携リンクの有効期限が切れました。どうすればよいですか？

連携トークンは **2 分後**に期限切れになります。`bind` を再度送信して新しいリンクを取得してください。

## 機能の使用

### Best 50 を生成するにはどうすればよいですか？

以下のいずれかのコマンドを送信：
- `b50`
- `best50`
- `ベスト50`

まず SEGA ID を連携していることを確認してください！

詳細は[スコアコマンド](/ja/commands/record)を参照してください。

### どのくらいの頻度でスコアを更新すべきですか？

プレイするたびに更新してください：
```
maimai update
```

これにより、maimai NET から最新のスコアが取得されます。

### スコア更新にはどのくらい時間がかかりますか？

通常 20 秒から 30 秒、以下によって異なります：
- プレイした楽曲数
- SEGA サーバーの応答時間
- キュー待ち時間（多くのユーザーが更新している場合）

### なぜ最新のスコアが表示されないのですか？

1. **更新しましたか？** まず `maimai update` を実行
2. **maimai NET を確認**: 公式サイトにスコアがありますか？
3. **待機時間**: スコアが maimai NET に表示されるまで数分かかる場合があります
4. **ネットワークの問題**: SEGA サーバーが遅い可能性があります

## 機能とコマンド

### フレンドシステムとは何ですか？

他の JiETNG ユーザーをフレンドとして追加して：
- Best 50 チャートを比較
- お互いのスコアを表示
- maimai コミュニティを構築

詳細は[フレンドシステム](/ja/features/friends)を参照してください。

## トラブルシューティング

### 「SEGA ID を連携していません」

**解決策**: `bind` コマンドでアカウントを連携してください。

[アカウント連携](/ja/guide/binding)を参照してください。

### 「スコアの更新に失敗しました」

**考えられる原因**:
- SEGA サーバーのダウン
- SEGA 認証情報が正しくない
- ネットワークタイムアウト
- maimai NET のメンテナンス

**解決策**:
- 数分待ってから再試行
- maimai NET に直接ログインして認証情報を確認
- SEGA のメンテナンス告知を確認
- 認証情報が変更された場合は再連携

### 「楽曲が見つかりません」

**原因**:
- 楽曲名の誤字
- 『maimai でらっくす』に楽曲が存在しない
- 間違ったバージョン（jp vs intl）

**解決策**:
- 別のキーワードを試す
- スペルを確認
- [maimai wiki](https://maimai.fandom.com/) で検索
- 英語名ではなく日本語名を試す（またはその逆）

### コマンドが機能しない

**トラブルシューティング手順**:
1. スペルと文法を確認
2. 連携済みか確認（`get me`）
3. スコアを更新（`maimai update`）
4. [コマンドリファレンス](/ja/commands/basic)を参照
5. まずシンプルなコマンドを試す（例：`b50 -lv 14` の前に `b50`）

## プライバシーとセキュリティ

### データは安全ですか？

はい！JiETNG は：
- ✅ SEGA 認証情報を暗号化
- ✅ 公開されている maimai NET データのみにアクセス
- ✅ チャット履歴を保存しません
- ✅ データ保護のベストプラクティスに従います

詳細は[プライバシーポリシー](/ja/more/privacy)を参照してください。

### データを削除するにはどうすればよいですか？

`unbind` を送信して、JiETNG からすべてのデータを永久に削除します。
