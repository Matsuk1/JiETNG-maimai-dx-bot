JiETNG は広範な開発作業の成果であり、重要な知的財産を表しています。独自ライセンスは以下を保証します：

1. **品質管理**：公式バージョンのみがサポートされます
2. **セキュリティ**：悪意のある変更を防ぎます
3. **持続可能性**：開発投資を保護します
4. **ユーザー保護**：すべてのユーザーが本物の安全なバージョンを受け取ることを保証
