# 楽曲検索とレコード照会

maimai でらっくす の楽曲情報を検索し、ランダム楽曲を取得し、完全な楽曲データベースを探索できます。

## 楽曲情報検索

楽曲名、略称、またはキーワードで検索し、詳細情報を取得します。

### 基本検索

**コマンド形式：**

```
[楽曲名] + info
[楽曲名] + song-info
[楽曲名] + ってどんな曲
```

**例：**

```
blew moon info
グリーンライツ・セレナーデ ってどんな曲
AMAZING MIGHTYYYY song-info
```

### 検索動作

- **あいまいマッチング**：スマートマッチング（類似度しきい値 85%）
- **複数結果表示**：最大 6 曲のマッチング楽曲を表示
- **部分名前マッチング**：「amazing might」で「AMAZING MIGHTYYYY!!!!!」がマッチ

:::tip 検索のヒント
- 英語または日本語の名称が使用可能
- フルネームと略称に対応
- 大文字小文字を区別しません
- 特殊記号は通常省略可能
:::

### 表示内容

各結果には以下が含まれます：

- 📝 **楽曲タイトル**（英語と日本語）
- 🎨 **カバーアート**
- 🎵 **アーティスト**
- 📅 **バージョン情報**
- 🎮 **利用可能な難易度**（Basic / Advanced / Expert / Master / Re:MASTER）
- 📊 **譜面定数**
- 🎯 **譜面タイプ**（Standard / でらっくす）
- 🎬 **カテゴリー（ジャンル）**

---

## ランダム楽曲

ランダムに楽曲を取得し、オプションでレベルを指定できます。

### 基本ランダム

**コマンド形式：**

```
random
random-song
ランダム
ランダム曲
```

**例：**

```
random
```

maimai でらっくす 全楽曲ライブラリからランダムに 1 曲抽出します。

### レベル指定ランダム

**コマンド形式：**

```
random [レベル]
random-song [レベル]
ランダム [レベル]
```

**例：**

```
random 14
ランダム曲 13+
random-song 15
```

### レベルフィルター構文

- `14` は 14.0～14.4 を表します
- `13+` は 13.5～13.9 を表します
- `14.6` は定数 14.6 の譜面のみを選択

:::tip ランダムチャレンジ
ランダム機能は以下に使用できます：
- 練習チャレンジ
- 新曲発見
- 慣れた曲風の打破
- デイリータスクの目標として
:::

## バージョン別楽曲表示

特定の maimai でらっくす バージョンで追加されたすべての楽曲を表示します。

**コマンド形式：**

```
[バージョン名] + version
[バージョン名] + version-list
[バージョン名] + のバージョンリスト
```

**例：**

```
FESTiVAL version
BUDDiES PLUS のバージョンリスト
Splash version-list
でらっくす PLUS version
```

:::tip ヒント
- `FESTiVAL+` は自動的に `FESTiVAL PLUS` として認識されます
:::

**表示内容：**
- 楽曲リスト

---

## スコア照会

特定の楽曲のプレイ記録を表示します。

**コマンド形式：**

```
[楽曲名] + record
[楽曲名] + song-record
[楽曲名] + のレコード
```

**例：**

```
blew moon record
オンゲキ音頭 のレコード
AMAZING MIGHTYYYY song-record
```

:::warning アカウント連携が必要
スコア機能を使用するには、まず SEGA ID を連携する必要があります。詳細は [アカウント連携](/ja/guide/binding) を参照
:::

**表示内容：**
- 📊 達成率
- 🎵 でらっくす スコア
- 🏆 完成ステータス（FC / FC+ / AP / AP+）
- 💎 シンクステータス（FS / FS+ / Fでらっくす / Fでらっくす+）
- 📈 レーティング貢献値

「レコードが見つかりません」と表示される場合：
- この曲をプレイしていない可能性
- スコアが更新されていない（`maimai update` を試してください）
- 名前マッチングエラー（まず info で検索してください）

## レベル別スコア表示

指定したレベルのすべてのスコアを表示します。

**コマンド形式：**

```
[レベル] + record-list
[レベル] + records
[レベル] + のレコードリスト
```

**例：**

```
14 record-list
13+ のレコードリスト
15 records
```

ページネーション：
```
14 record-list 2
13+ のレコードリスト 3
```

:::tip ヒント
レベルレコードで以下が可能：
- 高難易度の進捗追跡
- 改善の余地を見つける
- 弱点の分析
- 練習目標の設定
:::

## 高度な使用方法

### 類似楽曲を見つける

1. 好きな楽曲を検索：
   ```
   blew moon info
   ```
2. カテゴリーまたはアーティストを確認
3. 同じバージョンの曲目リストで類似作品を探す

### 新曲発見

1. レベル別ランダム：
   ```
   random 13+
   ```
2. 楽曲情報を確認
3. プレイ済みの場合、スコアを直接確認可能

### 練習計画

1. レベル曲目を確認：
   ```
   14 record-list
   ```
2. 改善が必要な曲を見つける
3. 譜面詳細を確認して練習をサポート

## 比較：検索 vs スコア照会

| 機能 | 楽曲情報検索 | スコア照会 |
|------|---------------|-----------|
| **目的** | 楽曲データ取得 | 個人スコア表示 |
| **連携必要** | ❌ 不要 | ✅ 必要 |
| **表示内容** | 曲目情報 | 個人データ |
| **用途** | 探索/楽曲理解 | 進捗追跡 |
| **応答速度** | 高速 | 高速（キャッシュ） |

:::tip プロのヒント
楽曲名が確実な場合、record コマンドを直接使用できます：
```
blew moon record
```
これにより楽曲情報とスコアの両方が表示されます！
:::

## トラブルシューティング

### 「楽曲が見つかりません」

**考えられる原因：**
- スペルミス
- 楽曲が maimai でらっくす に存在しない
- バージョン違い（JP / 国際版）

**解決策：**
- キーワードを変更
- [maimai wiki](https://maimai.fandom.com/) を確認
- 英語または日本語の名前を試す

### ランダム曲の重複

通常のランダム確率の範囲内です。レベルフィルターで重複を減らせます：
```
random 14+
```

### バージョンリストが不完全

**考えられる原因：**
- 名前が間違っている
- データベースの更新待ち

**解決策：**
- スペルを確認
- 別の表記を試す
- [GitHub Issues](https://github.com/Matsuk1/JiETNG/issues) で報告可能

## クイックリファレンス

```bash
# 検索
[song] info          # 楽曲情報を確認
[song] record        # 個人スコアを確認

# ランダム
random               # ランダムに 1 曲
random 14            # Lv14 をランダム

# バージョン
FESTiVAL version     # FESTiVAL の楽曲を表示

# レベル別スコア
14 record-list       # Lv14 楽曲のスコア
13+ records 2        # Lv13+ スコアの 2 ページ目
```
