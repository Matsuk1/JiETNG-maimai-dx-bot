# スコアシステム

Best 50（b50）とBest 100（b100）チャートは、JiETNGのフラッグシップ機能で、最高評価のスコアを美しくデザインされたビジュアライゼーションで表示します。

## Best 50とは？

「Best 50」システムは、maimai DXの公式ランキング方法で、以下で構成されています：

- **Best 35**：**旧曲（以前のバージョンの楽曲）**からのトップ35スコア
- **Best 15**：**新曲（現在のバージョンの楽曲）**からのトップ15スコア

あなたの **maimai DX Rating** は、これら50スコアの合計です。

## コマンド

### 基本的なBestチャート

```
b50          # Best 50チャートを生成
b100         # Best 100チャートを生成
best50       # b50の別名
best100      # b100の別名
```

### バリエーション

```
best35       # トップ35の旧曲のみ表示
best15       # トップ15の新曲のみ表示
ab50         # All Best 50（楽曲バージョンを無視）
ab35         # All Best 35
apb50        # All Perfect Best 50（AP/AP+スコアのみ）
idlb50       # Ideal Best 50（SSS+スコアをシミュレート）
```

## チャート機能

### 表示内容

各スコアカードには以下が表示されます：

- 🎵 **楽曲名**と難易度
- 🎮 **楽曲バージョン**：旧曲または新曲
- ⭐ **内部レベル**：実際の難易度（例：14.7）
- 💯 **達成率**：スコアのパーセンテージ
- 🏆 **スコアランク**：SSS+、SSS、SS+など
- 🌟 **フルコンボタイプ**：AP+、AP、FC+、FC
- 🔥 **フルシンク**：FDX+、FDX、FS+、FS
- 📊 **Rating**：個別楽曲のレーティング（0-22.4）
- 📈 **DXスコア**：あなたのDXスコア / 最大可能スコア

### ユーザー情報ヘッダー

チャートにはあなたのプロフィールが含まれます：

- 👤 プレイヤー名とアイコン
- 🏅 段位/クラスランク
- ⭐ 総合レーティング
- 🎖️ 称号プレート
- 📅 最終更新時刻

## 高度な使い方

### スコアのフィルタリング

b50出力をカスタマイズするためのフィルタを適用できます：

#### レベルでフィルタ

```
b50 -lv 15              # レベル15の楽曲のみ
b50 -lv 14 15           # レベル14-15
b50 -lv 13+             # レベル13.7+以上
```

#### Ratingでフィルタ

```
b50 -ra 200             # Rating 200+のみ
b50 -ra 180 200         # Rating 180-200
```

#### 達成率でフィルタ

```
b50 -scr 100.5          # 達成率100.5%+
b50 -scr 99 100         # 達成率99%-100%
```

#### DXスコアでフィルタ

```
b50 -dx 95              # DXスコア95%+
b50 -dx 90 95           # DXスコア90-95%
```

### フィルタの組み合わせ

複数のフィルタを組み合わせることができます：

```
b50 -lv 15 -scr 100.5            # レベル15で達成率100.5%+の楽曲
b50 -ra 200 -dx 95               # Rating 200+でDXスコア95%+
b50 -lv 14 15 -scr 99.5 -dx 90   # 複雑なフィルタ
```

## チャートタイプの説明

### Best 50 (b50)

公式ルールに従った標準ランキングチャート：
- 旧曲からのトップ35
- 新曲からのトップ15
- 実際のmaimai DX Rating

**使用例**：公式レーティングと進捗を確認

### Best 100 (b100)

より多くのスコアを表示する拡張バージョン：
- 旧曲からのトップ70
- 新曲からのトップ30
- レーティング楽曲を超えて確認

**使用例**：b50のしきい値をわずかに下回る楽曲を見つける

### All Best 50 (ab50)

楽曲バージョンの区別を無視：
- バージョンに関係なくトップ50スコア
- すべて旧曲またはすべて新曲の可能性あり
- 絶対的なベストプレイを表示

**使用例**：カテゴリに関係なく最高の実績を確認

### All Perfect Best 50 (apb50)

完璧なプレイのみを表示：
- AP（All Perfect）とAP+スコアのみ
- レーティングでランク付け
- パーフェクトコンボを追跡

**使用例**：パーフェクトプレイの進捗を監視

### Ideal Best 50 (idlb50)

理論上の最大レーティング：
- すべての楽曲でSSS+（100.5%+）をシミュレート
- 潜在的なレーティング上昇を表示
- 改善のモチベーション

**使用例**：レーティング向上の目標を設定

## 出力の理解

### Rating計算

各楽曲のレーティングは以下のように計算されます：

```
ベースRating = 内部レベル × 10
達成率修正値 = 達成率%に基づく

最終Rating = ベースRating + 達成率修正値
```

達成率修正値：
- **100.5000%+**：+2.0 rating (SSS+)
- **100.0000%**：+1.5 rating (SSS)
- **99.5000%+**：+1.0 rating (SS+)
- **99.0000%+**：+0.5 rating (SS)
- **98.0000%+**：+0.0 rating (S+)
- **97.0000%+**：-1.0 rating (S)
- 97%未満：より低いレーティング

### カラーコーディング

チャートはパフォーマンスを示す色を使用します：
- 🟣 **紫**：SSS+ (100.5%+)
- 🔴 **赤**：SSS/SS+ (99.5-100.49%)
- 🟠 **オレンジ**：SS/S+ (98-99.49%)
- 🟡 **黄色**：S (97-97.99%)
- ⚪ **グレー**：S未満

## ヒント＆トリック

### レーティングを最大化する

1. **高内部値の楽曲に集中**：レベル14.9、15.0の譜面
2. **SSS → SSS+に改善**：楽曲あたり+0.5レーティング
3. **SS+ → SSSに上げる**：+0.5レーティング増加
4. **低レーティングのb50楽曲を置き換える**：機会を見つけるためb100を確認

### 改善目標を見つける

フィルタを使用して具体的な目標を見つけます：

```
b100 -scr 99.5 100.49    # SSS楽曲を見つけてSSS+に
b100 -lv 15              # すべてのレベル15のプレイを確認
```

b50とab50を比較してバージョンの不均衡を確認：

```
b50     # 公式レーティング
ab50    # 全体的な最高スコア
```

ab50が大幅に高い場合は、過小評価されている楽曲バージョンに焦点を当てましょう。

## よくある質問

### ゲーム内と異なるレーティングが表示されるのはなぜ？

- JiETNGは`maimai update`を実行した時のみ更新されます
- 公式レーティングは各プレイ後に更新されます
- スコアを定期的に同期して最新の状態を保ちましょう

### 一部のスコアが表示されませんか？

最近楽曲をプレイしたことを確認してください。古いスコアは以下の場合表示されない可能性があります：
- より良いスコアに置き換えられた
- 楽曲がゲームから削除された
- データ同期の問題（再度`maimai update`を試してください）

### 過去のb50を見ることはできますか？

現在、JiETNGは現在のb50を表示します。履歴追跡は将来のアップデートで予定されています。
